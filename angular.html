Introduction

AngularJS is a very powerful JavaScript Framework. It is used in Single Page Application (SPA) projects. It extends HTML DOM with additional attributes and makes it more responsive to user actions. AngularJS is open source, completely free, and used by thousands of developers around the world

The main features of ANGULAR are as follows (but are not limited to) :

  • DATA-BINDING in AngularJS is the automatic synchronization of data between the model and view components. It works in a way that data-binding lets you treat the model as the single-source-of-truth in your application. The view is a projection of the model at all times. When the model changes, the view mirrors the change.

  • INPUT VALIDATION You can improve overall data quality by validating user input for accuracy and completeness.
    1. AngularJS offers client-side form validation.
    2. AngularJS monitors the state of the form and input fields (input, textarea, select), and lets you notify the user about the current state.
    3. AngularJS also holds information about whether they have been touched, or modified, or not.

  • FILTERS Filters can be added in AngularJS to format data.It provides filters to transform data:
    1. Date - Format a date to a specified format.
    2. Filter - Select a subset of items from an array.
    3. JSON - Format an object to a JSON string.
    4. Lowercase | Uppercase - Format a string to lower or upper case.

  • EVENTS The event directives allows us to run AngularJS functions at certain user events. An AngularJS event will not overwrite an HTML event, both events will be executed

Filters

          

          

            AngularJS provides filters to transform data. It ranges from currency, date, filter, 
            JSON, limitTo, lowercase/uppercase.They can also be used to filter data in an array. 
            This is a very powerful feature and is really helpful.Filters are used to modify the data.
             They can be clubbed in expression or directives using pipe (|) character.  
          

          
CODE

          
          
            
            <div ng-app="myApp" ng-controller="costCtrl">

            <h1>Price: {{ price | currency }}</h1>

            </div>
          

          
OUTPUT

          
Enter your earning:

          
          SALARY : {{ price | currency }}
          
EXPLANATION

          
The above code snippet is a filter that converts a number input from user into currency format. This is done as a '$' sign is included
            and thus symbolising a dollar value. Other than currency filter there are many other filters as described above.
